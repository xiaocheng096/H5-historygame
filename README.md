# 《山河破碎，觉醒之途：近代中国的磨难与抗争》H5互动故事

## 1. 项目概述

### 项目名称
山河破碎，觉醒之途

### 项目类型
H5互动历史教育故事

### 核心功能
基于《中国近现代史纲要》第一章的四节内容，通过交互式叙事让用户亲历近代中国的磨难与抗争历程，最终生成专属"觉醒纪念卡"。

---

## 2. 视觉与渲染规范

### 整体配色方案
- **主色调**：深红色 `#8B0000`（代表革命与热血）
- **辅助色**：黑色 `#1A1A1A`（代表沉重与反思）
- **背景色**：深灰色 `#2D2D2D`（传递历史厚重感）
- **强调色**：金色 `#D4AF37`（用于重要元素高亮）
- **文字色**：米白色 `#F5F5DC`（保证可读性）

### 字体选择
- 标题：Noto Serif SC（衬线体，庄重感）
- 正文：Noto Sans SC（无衬线，易读）
- 备用：serif / sans-serif

### 页面布局
- 全屏滑动式单页应用
- 每个章节占满一屏
- 移动端优先设计，适配375px-428px宽度
- 支持横屏与竖屏

### 视觉风格
- 庄重写实，略带水墨风格
- 旧照片质感滤镜
- 动态粒子效果（战火、尘埃）
- 渐变过渡动画

## 4. 功能与交互规范

### 4.1 开篇封面
- 标题动画淡入
- 背景：山河破碎的意象图（使用CSS渐变模拟）
- "开始历程"按钮，点击后进入第一章
- 背景音乐：低沉庄严的旋律（Web Audio API生成）

### 4.2 第一章：鸦片战争前后的中国与世界

#### 内容
- 对比地图动画：清王朝衰落 vs 西方殖民扩张
- 鸦片战争导火索互动

#### 互动：鸦片问题抉择
- 用户扮演清朝官员
- 三个选项：
  1. 严禁鸦片 → 进入"林则徐禁烟"剧情
  2. 放任自流 → 进入"鸦片泛滥"剧情
  3. 局部管制 → 进入"折中方案"剧情
- 每个选项触发不同的简短剧情描述
- 最终三条线汇合到鸦片战争爆发

#### 动画
- 地图边界线绘制动画
- 鸦片流入的流动粒子效果

### 4.3 第二章：西方列强对中国的侵略

#### 内容
- 时间轴展示不平等条约（南京条约、天津条约、马关条约、辛丑条约等）
- "多重压迫"互动

#### 互动：四重压迫
- 四个图标：军事侵略、政治控制、经济掠夺、文化渗透
- 点击任意图标，展开对应情景小故事
- 每个图标对应一个弹出层/抽屉

#### 视觉
- 深色、灰色为主的配色
- 时间轴使用垂直布局
- 压抑沉重的氛围

### 4.4 第三章：反抗外国武装侵略的斗争

#### 内容
- 甲午海战剧情
- 义和团运动剧情
- 民族英雄彩蛋

#### 互动：战斗抉择
- 用户扮演士兵参与战斗
- 关键节点选择：
  - 甲午海战：指挥舰队进攻/防守
  - 义和团：选择战斗/撤退
- 不同选择触发不同结局描述

#### 彩蛋：民族英雄
- 隐藏的"邓世昌"人物卡片
- 点击触发其生平介绍
- 解锁后显示金色边框效果

### 4.5 第四章：反侵略战争的失败与民族意识的觉醒

#### 内容
- 失败原因反思互动
- 民族觉醒时间线
- 觉醒纪念卡生成

#### 互动：失败原因反思
- 三个核心原因选项：
  1. 制度腐败
  2. 军事落后
  3. 思想僵化
- 选择后进入对应的科普剧情

#### 民族觉醒时间线
- 横向滑动时间轴
- 展示代表人物（魏源、严复、孙中山等）
- 展示思想主张

#### 纪念卡生成
- 用户输入姓名/昵称
- 用户输入个人感悟
- 自动生成主题："铭记历史，吾辈自强"
- 支持生成为图片（使用Canvas）
- 设计元素包括：
  - 深色背景配合暗红色渐变
  - 四角装饰性边框（金色）
  - 中心红色印章（带"覚"字）
  - 持卡人姓名展示区
  - 心得感悟文字框
  - 底部主题信息

### 4.6 分享功能
- 生成分享图片
- 复制分享链接
- 微信/朋友圈分享按钮（调用Web Share API）

## 5. 技术实现

### 技术栈
- 纯HTML5 + CSS3 + JavaScript
- 无外部框架依赖
- Canvas API用于纪念卡生成
- Web Audio API用于背景音乐
- LocalStorage存储进度

### 性能目标
- 首屏加载 < 2秒
- 页面切换流畅（60fps）
- 移动端电池友好

### 兼容性
- iOS Safari 12+
- Android Chrome 80+
- 微信内置浏览器

## 6. 内容大纲

### 第一章内容要点
- 鸦片战争前的中国：康乾盛世后的衰落
- 西方殖民扩张：工业革命后的英国
- 虎门销烟与战争爆发

### 第二章内容要点
- 南京条约（1842）：五口通商、割让香港
- 天津条约（1858）：公使进京、长江航行
- 马关条约（1895）：割让台湾、赔款白银
- 辛丑条约（1901）：使馆区、战争赔款

### 第三章内容要点
- 甲午海战（1894-1895）：邓世昌、黄海之战
- 义和团运动（1900）：扶清灭洋、八国联军
- 台湾抗日斗争

### 第四章内容要点
- 失败原因分析
- 魏源"师夷长技以制夷"
- 严复《天演论》
- 孙中山"振兴中华"
